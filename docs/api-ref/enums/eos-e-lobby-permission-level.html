EOS_ELobbyPermissionLevel

EOS API reference page for EOS_ELobbyPermissionLevel

Under 1 min to read

This enumerated type is part of the Lobby Interface.

Remarks

Permission level gets more restrictive further down

Values

EOS_ELobbyPermissionLevel

Enumerated Value Meaning
EOS_LPL_PUBLICADVERTISED Anyone can find this lobby as long as it isn't full
EOS_LPL_JOINVIAPRESENCE Players who have access to presence can see this lobby
EOS_LPL_INVITEONLY Only players with invites registered can see this lobby
