EOS_Sessions_JoinSessionOptions

EOS API reference page for EOS_Sessions_JoinSessionOptions

1 min to read

This data structure is part of the Sessions Interface.

Remarks

Input parameters for the EOS_Sessions_JoinSession function.

Properties

EOS_Sessions_JoinSessionOptions

Property Value
int32_t ApiVersion API Version: Set this to EOS_SESSIONS_JOINSESSION_API_LATEST.
const char* SessionName Name of the session to create after joining session
EOS_HSessionDetails SessionHandle Session handle to join
EOS_ProductUserId LocalUserId The Product User ID of the local user who is joining the session
EOS_Bool bPresenceEnabled If true, this session will be associated with presence. Only one session at a time can have this flag true. This affects the ability of the Social Overlay to show game related actions to take in the user's social graph. * using the bPresenceEnabled flags within the Sessions interface * using the bPresenceEnabled flags within the Lobby interface * using EOS_PresenceModification_SetJoinInfo
