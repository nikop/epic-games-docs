EOS_RTCAudio_GetAudioInputDevicesCount

EOS API reference page for EOS_RTCAudio_GetAudioInputDevicesCount

Under 1 min to read

Remarks

DEPRECATED! Use EOS_RTCAudio_QueryInputDevicesInformation, EOS_RTCAudio_GetInputDevicesCount instead. Returns the number of audio input devices available in the system. The returned value should not be cached and should instead be used immediately with the EOS_RTCAudio_GetAudioInputDeviceByIndex function.

Return Value

The number of audio input devices

See Also

EOS_RTCAudio_GetAudioInputDeviceByIndex, EOS_RTCAudio_AddNotifyAudioDevicesChanged
