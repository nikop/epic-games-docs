EOS_ELobbyAttributeVisibility

EOS API reference page for EOS_ELobbyAttributeVisibility

Under 1 min to read

This enumerated type is part of the Lobby Interface.

Remarks

Advertisement properties for a single attribute associated with a lobby

Values

EOS_ELobbyAttributeVisibility

Enumerated Value Meaning
EOS_LAT_PUBLIC Data is visible to lobby members, searchable and visible in search results.
EOS_LAT_PRIVATE Data is only visible to the user setting the data. Data is not visible to lobby members, not searchable, and not visible in search results.
