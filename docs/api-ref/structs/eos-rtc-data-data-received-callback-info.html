EOS_RTCData_DataReceivedCallbackInfo

EOS API reference page for EOS_RTCData_DataReceivedCallbackInfo

Under 1 min to read

Remarks

This struct is passed in with a call to EOS_RTCData_AddNotifyDataReceived registered event.

Properties

EOS_RTCData_DataReceivedCallbackInfo

Property Value
void* ClientData Client-specified data passed into EOS_RTCData_AddNotifyDataReceived.
EOS_ProductUserId LocalUserId The Product User ID of the user who initiated this request.
const char* RoomName The room associated with this event.
uint32_t DataLengthBytes The size of the data received. Max value is EOS_RTCDATA_MAX_PACKET_SIZE.
const void* Data The data received.
EOS_ProductUserId ParticipantId The Product User ID of the participant which sent the data.
