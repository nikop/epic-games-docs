EOS_ActiveSession_Info_Release

EOS API reference page for EOS_ActiveSession_Info_Release

Under 1 min to read

This function is part of the Sessions Interface.

Remarks

Release the memory associated with an EOS_ActiveSession_Info structure. This must be called on data retrieved from EOS_ActiveSession_CopyInfo.

See Also

EOS_ActiveSession_Info, EOS_ActiveSession_CopyInfo
