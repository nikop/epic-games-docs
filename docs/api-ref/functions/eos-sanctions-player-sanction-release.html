EOS_Sanctions_PlayerSanction_Release

EOS API reference page for EOS_Sanctions_PlayerSanction_Release

Under 1 min to read

This function is part of the Sanctions Interface.

Remarks

Release the memory associated with a player sanction. This must be called on data retrieved from EOS_Sanctions_CopyPlayerSanctionByIndex.

See Also

EOS_Sanctions_PlayerSanction, EOS_Sanctions_CopyPlayerSanctionByIndex

Parameters

EOS_Sanctions_PlayerSanction_Release

Parameter Type And Name Usage Information
EOS_Sanctions_PlayerSanction* Sanction - The sanction data to release.
