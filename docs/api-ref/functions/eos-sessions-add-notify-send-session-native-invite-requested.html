EOS_Sessions_AddNotifySendSessionNativeInviteRequested

EOS API reference page for EOS_Sessions_AddNotifySendSessionNativeInviteRequested

2 mins to read

This function is part of the Sessions Interface.

Remarks

Register to receive notifications about a session "INVITE" performed by a local user via the overlay. This is only needed when a configured integrated platform has EOS_IPMF_DisableSDKManagedSessions set. The EOS SDK will then use the state of EOS_IPMF_PreferEOSIdentity and EOS_IPMF_PreferIntegratedIdentity to determine when the NotificationFn is called.

must call EOS_Sessions_RemoveNotifySendSessionNativeInviteRequested to remove the notification.

Return Value

handle representing the registered callback

See Also

EOS_IPMF_DisableSDKManagedSessions, EOS_IPMF_PreferEOSIdentity, EOS_IPMF_PreferIntegratedIdentity
