EOS_LobbyModification_SetPermissionLevel

EOS API reference page for EOS_LobbyModification_SetPermissionLevel

Under 1 min to read

This function is part of the Lobby Interface.

Remarks

Set the permissions associated with this lobby. The permissions range from "public" to "invite only" and are described by EOS_ELobbyPermissionLevel

Return Value

  • EOS_Success if setting this parameter was successful

  • EOS_IncompatibleVersion if the API version passed in is incorrect

Parameters

EOS_LobbyModification_SetPermissionLevel

Parameter Type And Name Usage Information
EOS_HLobbyModification Handle
const EOS_LobbyModification_SetPermissionLevelOptions* Options Options associated with the permission level of the lobby
