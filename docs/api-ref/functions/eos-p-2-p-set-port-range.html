EOS_P2P_SetPortRange

EOS API reference page for EOS_P2P_SetPortRange

Under 1 min to read

This function is part of the P2P Interface.

Remarks

Set configuration options related to network ports.

Return Value

  • EOS_EResult::EOS_Success - if the options were set successfully

  • EOS_EResult::EOS_InvalidParameters - if the options are invalid in some way
