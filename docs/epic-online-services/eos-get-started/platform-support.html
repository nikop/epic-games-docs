Platforms Supported

Lists the platforms and platform versions which the Epic Online Services (EOS) SDK supports.

1 min to read

These subsections list the platforms and platform versions which the Epic Online Services (EOS) SDK supports. See the System Requirements documentation for a lists of the operating systems and supporting software required to develop and ship your game on each supported platform.

Tip: You can call the EOS_GetVersion function to get the EOS SDK version and the version of the platform that your game is compiled against.

Platforms

PC

Windows

From April 2024 onwards, the EOS SDK will stop supporting Windows 7, Windows 8, and Windows 8.1.

EOS SDK versions released after April 2024 will not support these versions of Windows.

To use versions of the EOS SDK released after April 2024, you will need to update to a more recent version of Windows. See the EOS SDK for Windows documentation for more information.

The table below lists the Windows release version which each EOS SDK release version supports:

EOS SDK Release Version Windows Release Version
Any release version supports Windows 7 or higher

See the EOS SDK for Windows documentation for more information on how to use the EOS SDK for Windows.

MacOS

The table below lists the macOS release version which each EOS SDK release version supports:

EOS SDK Release Version MacOS Release Version
EOS SDK 1.14.1 - 1.16.1 supports MacOSX 10.12 or higher
EOS SDK 1.10.2 - 1.14 supports MacOSX 10.14 or higher

See the EOS SDK for Mac documentation for more information on how to use the EOS SDK for macOS.

Linux

All EOS SDK release versions support any modern Linux version that uses glibc version 2.17 or higher. The table below lists the Linux toolchain version which each EOS SDK release version supports:

EOS SDK Release Version Linux Toolchain Version
EOS SDK 1.11 - 1.16.1 supports v19_clang-11.0.1-centos7 or higher
EOS SDK 1.10.2 - 1.10.3 supports v16_clang-9.0.1-centos7 or higher

See the EOS SDK for Linux documentation for more information on how to use the EOS SDK for Linux.

Mobile

iOS

The table below lists the iOS release version which each EOS SDK release version supports:

EOS SDK Release Version iOS Release Version
EOS SDK 1.10.2 - 1.16.1 supports iOS 11 or higher

See the EOS SDK for iOS documentation for more information on how to use the EOS SDK for iOS.

Android

The table below lists the Android release version which each EOS SDK release version supports:

EOS SDK Release Version Android Release Version
EOS SDK 1.10.2 - 1.16.1 supports 6.0 (API level 23)

See the EOS SDK for Android documentation for more information on how to use the EOS SDK for Android.

Consoles

For a list of the console release versions that each EOS SDK release version supports, see the the documentation for each console. You can find this under the EOS SDK for Platforms section of this manual.

You can only access console documentation if you have the appropriate permissions. See the Get Started Steps: EOS SDK Download Types documentation for more information on how to access the EOS SDKs for consoles and their associated documentation.
