About the KWS Platform

An overview of the main developer features of Kids Web Services.

Under 1 min to read

This section provides an overview of the main developer features of the KWS platform:

  • The KWS Developer Portal, a simple web-based self-serve management console that enables you to set up the PV service on your product(s).
  • The KWS API, an HTTPS API that enables you to integrate easily with the PV service.
  • KWS webhooks. KWS uses these to notify you of important events.
  • Localization and language options in KWS.
