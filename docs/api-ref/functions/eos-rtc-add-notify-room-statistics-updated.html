EOS_RTC_AddNotifyRoomStatisticsUpdated

EOS API reference page for EOS_RTC_AddNotifyRoomStatisticsUpdated

2 mins to read

This function is part of the RTC Interface.

Remarks

Register to receive notifications to receiving periodical statistics update. If the returned NotificationId is valid, you must call EOS_RTC_RemoveNotifyRoomStatisticsUpdated when you no longer wish to have your StatisticsUpdateHandler called.

Return Value

Notification ID representing the registered callback if successful, an invalid NotificationId if not

See Also

EOS_INVALID_NOTIFICATIONID, EOS_RTC_RemoveNotifyRoomStatisticsUpdated

Parameters

EOS_RTC_AddNotifyRoomStatisticsUpdated

Parameter Type And Name Usage Information
EOS_HRTC Handle
const EOS_RTC_AddNotifyRoomStatisticsUpdatedOptions* Options
void* ClientData Arbitrary data that is passed back in the StatisticsUpdateHandler
const EOS_RTC_OnRoomStatisticsUpdatedCallback StatisticsUpdateHandler

This function enables your callback (the StatisticsUpdateHandler parameter) to respond to events that happen on the back-end service. Remember to remove your callback function when it is no longer needed.

Callback Function Information

Because this function is asynchronous, it employs a callback of type EOS_RTC_OnRoomStatisticsUpdatedCallback to report the results of its operation.

Callback Parameters

EOS_RTC_OnRoomStatisticsUpdatedCallback

Parameter Type And Name Usage Information
const EOS_RTC_RoomStatisticsUpdatedInfo* Data This struct is passed in with a call to EOS_RTC_AddNotifyRoomStatisticsUpdated registered event.
