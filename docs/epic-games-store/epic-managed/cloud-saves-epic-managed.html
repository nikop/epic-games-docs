Cloud Saves

Enable and test Cloud Save for your Epic-managed product

Epic-managed distribution on the Epic Games Store
2 mins to read

This article covers functionality for products using Epic-managed distribution. If the options described are not available to you, you might be using self-managed distribution. For documentation applicable to your experience, refer to the Epic Games Store section in Developer Resources.

The Cloud Save feature enables users to continue their game progress whenever and wherever they play next by automatically storing their save files in the cloud. This guide explains how to set up Cloud Saves for your product and provides test cases to ensure the feature is working as expected.

Cloud Saves Setup

Cloud Saves are enabled by default for users, however, they must be configured by your Release Manager for each product before you begin testing. Your Release Manager will configure the cloud saves on your Staging artifact so that you can test the feature. You need to provide your Release Manager with the following information:

  • Save folder location: This is where your game saves locally on a user's machine. If your game runs on both Mac and Windows, provide both locations.
  • Exclusions: Any files that you want to exclude from the cloud save.
  • Maximum folder size: The maximum size of the save folder. If exceeded, a user gets the error "The save files are too large to upload." Learn more: Cloud Save Error Codes

To remove Cloud Save support from your product, contact your Release Manager.

Cloud Saves Testing and Error Codes

To ensure that Cloud Saves are working optimally for your product, carry out our recommended Cloud Saves Test Cases. These test cases include instructions and expected results for each scenario.

Learn more:

  • Cloud Saves Test Cases
  • Potential error scenarios and their codes: Cloud Saves Error Codes.

If you need help with any other Cloud Saves testing issues that are not covered in our test cases, contact your Release Manager.

When you finish testing Cloud Saves on your Staging artifact, let your Release Manager know so they can enable the Cloud Saves on the Production artifact.

Cloud Saves conflicts in the Epic Games Launcher

A user of your product might run into Cloud Saves conflicts if they use your product on more than one device. If this happens, they can resolve the conflict with the following Epic Games online support page: How do I handle Cloud Save conflicts in the Epic Games Launcher?
