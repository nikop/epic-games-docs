EOS_EApplicationStatus_ToString

EOS API reference page for EOS_EApplicationStatus_ToString

Under 1 min to read

Remarks

Gets the string representation of an EOS_EApplicationStatus value. Example: EOS_EApplicationStatus_ToString(EOS_EApplicationStatus::EOS_AS_Foreground) returns "EOS_AS_Foreground".

Return Value

  • Pointer to a static string representing the input enum value.

  • The returned string is guaranteed to be non-null, and must not be freed by the application.

Parameters
