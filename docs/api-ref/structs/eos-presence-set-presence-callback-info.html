EOS_Presence_SetPresenceCallbackInfo

EOS API reference page for EOS_Presence_SetPresenceCallbackInfo

Under 1 min to read

This data structure is part of the Presence Interface.

Remarks

The result meta-data from setting a user's presence.

Properties

EOS_Presence_SetPresenceCallbackInfo

Property Value
EOS_EResult ResultCode Result code for the operation. EOS_Success is returned if presence was successfully set, other codes indicate an error
void* ClientData Client-specified data passed into EOS_Presence_SetPresence
EOS_EpicAccountId LocalUserId The Epic Account ID of the local user that had their presence set
