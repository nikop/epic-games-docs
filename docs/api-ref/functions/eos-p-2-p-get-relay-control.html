EOS_P2P_GetRelayControl

EOS API reference page for EOS_P2P_GetRelayControl

Under 1 min to read

Google has informed developers of a vulnerability in versions (earlier than M102) of WebRTC. Impacts, workarounds, and updates can be found here.

This function is part of the P2P Interface.

Remarks

Get the current relay control setting.

Return Value

  • EOS_EResult::EOS_Success - if the input was valid

  • EOS_EResult::EOS_InvalidParameters - if the input was invalid in some way
