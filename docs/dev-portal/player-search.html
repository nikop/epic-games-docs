EOS Player Search

Player search, linked accounts, user data retrieval, and user data deletion.

6 mins to read

Use the Player Search page to view and manage the data that Epic Online Services (EOS) stores about your product users on your behalf.

Note: For information on the ticketing system (known as "player ticketing") to manage support for users of your product, see the following documentation: Ticketing System.

The Player Search Page

To access the Player Search page:

  1. In the Developer Portal (dev.epicgames.com/portal), make sure you are in the correct organization relating to your product.
    To do this:
    • a) Select Organization from the top of the left navigation’s dropdown menu (in the image above, the organization name is “Your…”).
    • b) Follow the onscreen instructions to choose your organization.
  2. From the left navigation, click Dashboard to see the Developer Portal home page, and on that page, click Player Search.

On the Player Search page, you can search users through different user ID types. These are:

  • Product User ID - PUID (Glossary: ProductUserId )
  • Player Account ID - a unique ID from an external platform, for example, a user's Epic Games account ID or Steam ID.
  • Epic Account for development - an Epic account linked with a console development account.

The search results provide the following data about a user:

  • Player Profile
  • Linked Accounts
  • Data Rights

You can export or delete all stored data about a user.

Player Profile

In the Player Profile tab, you can see user-related EOS features that are enabled for your product and the user data for each feature. The features are:

  • Achievements
  • Stats
  • Player Data Storage
  • Anti-cheat
  • Sanctions

Linked Accounts

The Linked Accounts tab displays any external accounts a user has linked to their Product User ID using the EOS Connect Interface (documentation: Connect Interface). This information is useful for your organization's user or player support. Use it to resolve user or player issues, such as unlinking an accidentally-linked external account. Here you can also review the full account linking history for a user.

Note: The Linked Accounts tab can also give options for console-development accounts. For more information, see the Console Development Accounts section below.

Unlink a User's Account

Users might want to change which external accounts they have linked through the EOS Connect Interface. The following are examples of why users might want to do this:

  • The user no longer wants to maintain an account for a particular identity provider.
  • The user wants to link a different identity provider account with a different username.
  • You have to unlink a user's account with a specific identity provider for security reasons or fraud-prevention measures.

To remove a linked account for an identity provider, click Unlink Account.

Notes:

  • Once you unlink an account, your organization no longer tracks it, and the user can no longer sign in to your products using that identity provider unless they re-link.
  • When you unlink an account, it is instantaneous but unlinking does not necessarily remove other linked accounts or remove the user from the EOS service.
  • If you unlink an external account from an Epic Games account, and you try to unlink either the last-linked account for a user, or the only linked account for a user, you see a warning. The warning alerts you that you cannot unlink the user's only sign-in method.

Console Development Accounts

If your product uses Epic Account Services , when you sign up your product for console development with EOS, your console-development account is associated with new console-Epic-Games account. When you search for an Epic Games account using Player Search, the results return the new console-Epic-Games account as well as the console-development accounts associated with it.

In the Linked Accounts tab, you can delete the console-Epic-Games account. You can use this to test the Epic Games accounts sign-in flow again, re-using the same console-development account.

Note: When you have a console-Epic-Games account, the following applies:

  • You cannot unlink its console-development account.
  • Access to your console-Epic-Games account has restrictions:
    • By default, you cannot access the console-Epic-Games account outside of the console device.
      This is because you cannot use the linked console-development account to sign-in outside the console development environment.
    • You can amend this as follows:
      • When you sign into Epic Games on a web browser using a console-development account, the sign-in prompts you to assign an email and a password to the new console-Epic-Games account.
        After you do this, you can access the console-Epic-Games account outside of the console development environment. This enables you to connect additional external accounts to the console-Epic-Games account.
        Tip: Connect accounts in Epic Games's account management website (go to your account icon, top right of any page in the Developer Portal > Manage Account > Account Settings > APPS AND ACCOUNTS).
      • After you have assigned a username and password to the console-Epic-Games account, you can only unlink the console-Epic-Games account from your console-development account. You cannot delete the console-Epic-Games account itself.

For information on how to become a console developer with EOS, see the following documentation: EOS Get Started Steps: Access the SDK for Consoles.

Data Rights

Use the Data Rights tab to export or delete data stored for a user by EOS on your behalf. You can also export and delete user data for a specific product, if a user has used multiple products created by your organization.

Note: You cannot export data while a delete is in progress and you cannot delete data while an export is in progress.

Export Player Data

There are two export requests. These are as follows:

  • Export Product Data

    • To request a user's player progression and user data for a specific product, click Export Product Data for any product. (Note that this does not include organization data associated with that user ID.)
    • This request is linked with the user's Product User ID (PUID) which is unique to each of your products (Glossary: ProductUserId ).

  • Export All Data

    • To request a user's player progression and user data for all products your organization manages, including all data associated with a user ID, click Export All Data.

After you have made either request, EOS emails you, the requester at your organization, a link to a bundle containing the data. This bundle is accessible for 30 days, after which you must make a new request to obtain up-to-date data. You can issue requests to export data for a user once every 10 days. The maximum expected time to process a request is 14 days, though requests are usually processed more quickly.

The bundle might contain encrypted information. See the following documentation for guidance:

  • Player Data Storage
  • Title Storage
  • File Decryption Tool - for information on how to decrypt data.

Delete Player Data

Some circumstances might require deleting a user's data completely. For example:

  • You want to permanently remove a user due to a violation.
  • A user's account data might become corrupted or compromised.
  • A user might specifically request removal of their account data.

There are three deletion options:

  • Delete Product Data

    • To permanently delete a user's player progression and user data for that product, click Delete Product Data.

  • Delete User

    • To delete a user's data across all products managed by your organization, click Delete User at the top of the page.
    • Delete User also unlinks all their accounts for all identity providers and removes any organization data about the user not explicitly associated with any single product. This request has a 10-day grace period in case the deletion request is canceled. During the grace period, the account remains searchable, but the user won’t be able to sign in through EOS Connect.

  • Delete Product Data

    • To delete a user's player progression data on a single product, click Delete Product Data.
    • Be aware that linked accounts and organization-level data associated with that user are not affected by this delete and remain stored.
