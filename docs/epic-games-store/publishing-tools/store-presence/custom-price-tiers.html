Create Custom Price Tiers

Create price points outside default suggested pricing

4 mins to read

Are you a Closed Beta partner? If you are not participating in the Closed Beta release, you can access documentation by using your Epic Games account to sign in to the Knowledge Base.

The Custom Price Tiers page allows users to create and set price points outside the Epic Games Store default suggested price settings. Custom price tiers are reviewed and approved by Epic Games before going live.

  • The USD equivalent is calculated from the local price, according to a current exchange rate; the actual rate at the moment of purchase may vary. The final price is regulated by Epic Games.

Create a Custom Price Tier

  1. Go to Product > Custom Price Tiers > CREATE PRICE TIER.

  2. Enter a Tier name.

  3. Select the Base price tier.

  4. Enter the customized price for each region.

  5. SAVE to continue edits later, or SAVE AND SUBMIT for review. After approval, new or modified prices will be available as options in the offers section. Any changes will affect Live products associated with the price.
