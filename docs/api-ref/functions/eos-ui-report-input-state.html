EOS_UI_ReportInputState

EOS API reference page for EOS_UI_ReportInputState

1 min to read

This function is part of the UI Interface.

Remarks

Pushes platform agnostic input state to the SDK. The state is passed to the EOS Overlay on console platforms. This function has an empty implementation (i.e. returns EOS_NotImplemented) on all non-consoles platforms.

Return Value

  • EOS_Success If the Social Overlay has been notified about the request.

  • EOS_IncompatibleVersion if the API version passed in is incorrect.

  • EOS_NotConfigured If the Social Overlay is not properly configured.

  • EOS_ApplicationSuspended If the application is suspended.

  • EOS_NotImplemented If this function is not implemented on the current platform.
