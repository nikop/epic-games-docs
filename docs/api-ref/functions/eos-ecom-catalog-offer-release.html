EOS_Ecom_CatalogOffer_Release

EOS API reference page for EOS_Ecom_CatalogOffer_Release

Under 1 min to read

This function is part of the Ecom Interface.

Remarks

Release the memory associated with an EOS_Ecom_CatalogOffer structure. This must be called on data retrieved from EOS_Ecom_CopyOfferByIndex.

See Also

EOS_Ecom_CatalogOffer, EOS_Ecom_CopyOfferByIndex

Parameters

EOS_Ecom_CatalogOffer_Release

Parameter Type And Name Usage Information
EOS_Ecom_CatalogOffer* CatalogOffer - The catalog offer structure to be released
