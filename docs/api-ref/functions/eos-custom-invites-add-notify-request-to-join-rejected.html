EOS_CustomInvites_AddNotifyRequestToJoinRejected

EOS API reference page for EOS_CustomInvites_AddNotifyRequestToJoinRejected

3 mins to read

This function is part of the CustomInvites Interface.

Remarks

Register to receive notifications when a Request to Join for any logged in local user is rejected via the Social Overlay

must call EOS_CustomInvites_RemoveNotifyRequestToJoinRejected to remove the notification

Return Value

handle representing the registered callback

Parameters

EOS_CustomInvites_AddNotifyRequestToJoinRejected

Parameter Type And Name Usage Information
EOS_HCustomInvites Handle
const EOS_CustomInvites_AddNotifyRequestToJoinRejectedOptions* Options Structure containing information about the request.
void* ClientData Arbitrary data that is passed back to you in the CompletionDelegate.
const EOS_CustomInvites_OnRequestToJoinRejectedCallback NotificationFn A callback that is fired when a Request to Join is accepted via the Social Overlay.

This function enables your callback (the NotificationFn parameter) to respond to events that happen on the back-end service. Remember to remove your callback function when it is no longer needed.

Callback Function Information

Because this function is asynchronous, it employs a callback of type EOS_CustomInvites_OnRequestToJoinRejectedCallback to report the results of its operation. See the EOS_CustomInvites_OnRequestToJoinRejectedCallbackInfo page for more details, or check the [Callback Data] section below.
