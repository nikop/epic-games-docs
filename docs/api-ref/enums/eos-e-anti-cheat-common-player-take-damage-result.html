Remarks

The result of a damage event, if any

Values

EOS_EAntiCheatCommonPlayerTakeDamageResult

Enumerated Value Meaning
EOS_ACCPTDR_None No direct state change consequence for the victim
EOS_ACCPTDR_Downed Player character is temporarily incapacitated and requires assistance to recover
EOS_ACCPTDR_Eliminated Player character is permanently incapacitated and cannot recover (e.g. dead)
