EOS_RTCAudio_InputDeviceInformation_Release

EOS API reference page for EOS_RTCAudio_InputDeviceInformation_Release

Under 1 min to read

Remarks

Release the memory associated with EOS_RTCAudio_InputDeviceInformation. This must be called on data retrieved from EOS_RTCAudio_CopyInputDeviceInformationByIndex.

See Also

EOS_RTCAudio_InputDeviceInformation, EOS_RTCAudio_CopyInputDeviceInformationByIndex

Parameters

EOS_RTCAudio_InputDeviceInformation_Release

Parameter Type And Name Usage Information
EOS_RTCAudio_InputDeviceInformation* DeviceInformation - The audio input device's information to release.
