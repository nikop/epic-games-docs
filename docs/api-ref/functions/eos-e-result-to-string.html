EOS_EResult_ToString

EOS API reference page for EOS_EResult_ToString

Under 1 min to read

Remarks

Returns a string representation of an EOS_EResult. The return value is never null. The return value must not be freed. Example: EOS_EResult_ToString(EOS_Success) returns "EOS_Success"
