EOS_Platform_Create

EOS API reference page for EOS_Platform_Create

Under 1 min to read

Remarks

Create a single Epic Online Services Platform Instance. The platform instance is used to gain access to the various Epic Online Services. This function returns an opaque handle to the platform instance, and that handle must be passed to EOS_Platform_Release to release the instance.

Return Value

An opaque handle to the platform instance.
