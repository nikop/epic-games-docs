EOS_Lobby_ParseConnectString

EOS API reference page for EOS_Lobby_ParseConnectString

1 min to read

This function is part of the Lobby Interface.

Remarks

Parse the ConnectString for an EOS lobby invitation to extract just the lobby ID. Used for joining a lobby from a connection string (as generated by GetConnectString) found in a platform invitation or presence.

Return Value

  • EOS_Success if retrieving the string was successful.

  • EOS_InvalidParameters if the OutBuffer or InOutBufferLength are null.

  • EOS_IncompatibleVersion if the API version passed in is incorrect.

  • EOS_LimitExceeded if the provided InOutBufferLength is too small to contain the resulting string.

Parameters

EOS_Lobby_ParseConnectString

Parameter Type And Name Usage Information
EOS_HLobby Handle
const EOS_Lobby_ParseConnectStringOptions* Options Structure containing the input parameters. API version and ConnectString.
char* OutBuffer The buffer to store the null-terminated lobby ID within
uint32_t* InOutBufferLength In: The maximum amount of writable chars in OutBuffer see EOS_LOBBY_PARSECONNECTSTRING_BUFFER_SIZE, Out: The minimum amount of chars needed in OutBuffer to store the LobbyID (including the null-terminator). May be set to zero depending on the error result.
