EOS_RTCAudio_CopyOutputDeviceInformationByIndex

EOS API reference page for EOS_RTCAudio_CopyOutputDeviceInformationByIndex

1 min to read

Remarks

Fetches an audio output device's information from then given index that are cached locally.

Return Value

  • EOS_Success if the information is available and passed out in OutOutputDeviceInformation

  • EOS_InvalidParameters if you pass a null pointer for the out parameter

  • EOS_NotFound if the audio output device's information is not found

See Also

EOS_RTCAudio_OutputDeviceInformation_Release, EOS_RTCAudio_GetAudioOutputDevicesCount, EOS_RTCAudio_AddNotifyAudioDevicesChanged

Parameters

EOS_RTCAudio_CopyOutputDeviceInformationByIndex

Parameter Type And Name Usage Information
EOS_HRTCAudio Handle
const EOS_RTCAudio_CopyOutputDeviceInformationByIndexOptions* Options structure containing the index being accessed
EOS_RTCAudio_OutputDeviceInformation** OutOutputDeviceInformation The audio output device's information for the given index, if it exists and is valid, use EOS_RTCAudio_OutputDeviceInformation_Release when finished

If successful, this function provides data to the caller through an output parameter. Once you are finished with the data, you must release it by making the appropriate call into the EOS SDK.
