EOS_ENetworkStatus_ToString

EOS API reference page for EOS_ENetworkStatus_ToString

Under 1 min to read

Remarks

Gets the string representation of an EOS_ENetworkStatus value. Example: EOS_ENetworkStatus_ToString(EOS_ENetworkStatus::EOS_NS_Online) returns "EOS_NS_Online".

Return Value

  • Pointer to a static string representing the input enum value.

  • The returned string is guaranteed to be non-null, and must not be freed by the application.
