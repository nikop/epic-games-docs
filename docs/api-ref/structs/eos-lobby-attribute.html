EOS_Lobby_Attribute

EOS API reference page for EOS_Lobby_Attribute

Under 1 min to read

This data structure is part of the Lobby Interface.

Remarks

An attribute and its visibility setting stored with a lobby. Used to store both lobby and lobby member data

Properties

EOS_Lobby_Attribute

Property Value
int32_t ApiVersion API Version: Set this to EOS_LOBBY_ATTRIBUTE_API_LATEST.
EOS_Lobby_AttributeData* Data Key/Value pair describing the attribute
EOS_ELobbyAttributeVisibility Visibility Is this attribute public or private to the lobby and its members
