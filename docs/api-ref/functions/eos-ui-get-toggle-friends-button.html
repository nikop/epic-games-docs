EOS_UI_GetToggleFriendsButton

EOS API reference page for EOS_UI_GetToggleFriendsButton

Under 1 min to read

This function is part of the UI Interface.

Remarks

Returns the current Toggle Friends Button. This button can be used by the user to toggle the friends overlay when available. The default value is EOS_UISBF_None.

Return Value

  • A valid button combination which represents any number of buttons.

  • EOS_UIK_None will be returned if any error occurs.
