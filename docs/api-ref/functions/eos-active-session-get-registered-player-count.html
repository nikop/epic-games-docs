EOS_ActiveSession_GetRegisteredPlayerCount

EOS API reference page for EOS_ActiveSession_GetRegisteredPlayerCount

Under 1 min to read

This function is part of the Sessions Interface.

Remarks

Get the number of registered players associated with this active session

Return Value

number of registered players in the active session or 0 if there is an error
