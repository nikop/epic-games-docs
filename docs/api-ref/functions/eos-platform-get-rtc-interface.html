EOS_Platform_GetRTCInterface

EOS API reference page for EOS_Platform_GetRTCInterface

Under 1 min to read

Remarks

Get a handle to the Real Time Communications Interface (RTC). From the RTC interface you can retrieve the handle to the audio interface (RTCAudio), which is a component of RTC.

Return Value

EOS_HRTC handle

See Also

EOS_RTC_GetAudioInterface, eos_rtc.h, eos_rtc_types.h
