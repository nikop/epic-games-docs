Epic Online Services (EOS) Overview

Epic Online Services (EOS) are free, cross-platform services for multiplayer games, providing quick-to-install, high-quality game launching, operation, and scaling.

Epic Online Services (EOS) are free, cross-platform services for multiplayer games which you can use in your game whatever game engine you use for development, and whatever platform your game clients run on. EOS encompasses Game Services and Epic Account Services. EOS also provides services for the Epic Games Store.

Find out more about each set of services in the documentation on:

  • EOS Game Services
  • EOS Epic Account Services
  • Epic Games Store Services

EOS has several tools, including:

  • The Developer Portal.
    Find out more in the Developer Portal documentation.
  • The EOS Web API - HTTP APIs for common use cases.
    Find out more in the EOS Web API Overview and EOS Web API Reference documentation.
  • The EOS SDK - in C and C#.
    Find out more in the EOS SDK Key Information and EOS SDK API Reference documentation.
