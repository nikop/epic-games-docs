Profile

Details on what a player can do when viewing another player's profile.

2 mins to read

This documentation assumes that you install the Social Overlay with your game and initialize the EOS SDK when a player starts your game. For more information on how to install and initialize the Social Overlay, see the SDK Integration documentation.

On the Friends or Add Friends tab, a player can select their own Epic Games account or another player's to see the profile of that player. For more information on the Friends tab, see the Friends Tab documentation.

When a player opens a profile, the information they see depends on the version of the EOS SDK that you use for your game. See the sections below on each SDK version for more information.

Profile with EOS SDK 1.14

When you use EOS SDK 1.14 for your game, the player can use the profile to do the following:

  • Select INVITE to send an invitation to another player to join their game.

    Note: A player can only send a game invitation when they play your game (EOS SDK 1.14 only).

  • Select ACCEPT INVITE to join another player's game (if the player has received an invitation).
    Note: A player can only receive a game invitation and accept it when they play your game (EOS SDK 1.14 only).

Profile with EOS SDK 1.15

When you use EOS SDK 1.15 for your game, the player can use the Friends tab to do the following:

  • Select ADD EPIC FRIEND to send an invitation to another player to join their friends list.
  • Select ACCEPT EPIC FRIEND to add themselves to another player's friends list (if they have received the invitation).
  • Select INVITE to send an invitation to another player to join their game.
  • Select ACCEPT GAME INVITE to join another player's game (if they have received the invitation).
  • Select MORE OPTIONS > BLOCK to remove another player from their friends list (if they are on it). This stops them from receiving invitations from that player and hides their online status from the other player.

Note: If the player is under the age of digital consent and their account requires a Guardian PIN, then they have to enter the Guardian PIN to send or accept invitations. See the Epic Games Cabined Accounts documentation for more information.

Profile with EOS SDK 1.16

When you use EOS SDK 1.16 for your game, the player can use the Friends tab to do the following:

  • Select ADD EPIC FRIEND to send an invitation to another player to join their friends list.
  • Select ACCEPT EPIC FRIEND to add themselves to another player's friends list (if they have received the invitation).
  • Select INVITE to send an invitation to another player to join their game.
  • Select ACCEPT INVITE to join another player's game (if they have received the invitation).
  • Select MORE OPTIONS > BLOCK to remove another player from their friends list (if they are on it). This stops them from receiving invitations from that player, and hides their online status from the other player.
  • Select REQUEST TO JOIN to send a request to that player to join their game. They can join if that player accepts their request. See the Request to Join documentation for more information.
  • Select JOIN to join that player's game if the lobby or session allows join via presence.

Note: If the player is under the age of digital consent and their account requires a Guardian PIN, then they have to enter the Guardian PIN to send or accept invitations. See the Epic Games Cabined Accounts documentation for more information.
