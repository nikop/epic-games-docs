Firewall Considerations

A list of the host addresses that you need to access to use EOS SDK.

Under 1 min to read

The EOS SDK APIs and Web APIs connect to services hosted by Epic Online Services (EOS). All API connections use the HTTPS protocol (port 443).

Ensure your local network, router, and firewall configurations allow access to the host addresses listed below.

Additional information:

  • For details of the hardware and software required for your development environment and games client, see the documentation on System Requirements.
  • For details of the platforms and platform versions which the Epic Online Services (EOS) SDK supports, see the Platforms Supported documentation.

EOS Host Addresses

Your local network must be able to reach the following host addresses to use Epic Online Services:

  • api.epicgames.dev (EOS SDK API, EOS Web API)
  • epic-social-game-overlay-prod.ol.epicgames.com (EOS Social Overlay)
  • epicgames.com (Website)
  • dev.epicgames.com (Developer Portal, Documentation, Epic Games Store support)
  • eoshelp.epicgames.com (Epic Online Services support)

Ecommerce Host Addresses

Your local network must be able to reach the following host addresses to use ecommerce features, such as in-game microtransactions:

  • catalog-public-service-prod06.ol.epicgames.com
  • ecommerceintegration-public-service-ecomprod02.ol.epicgames.com
  • priceengine-public-service-ecomprod01.ol.epicgames.com
