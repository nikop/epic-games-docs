EOS_RTC_SetSetting

EOS API reference page for EOS_RTC_SetSetting

Under 1 min to read

This function is part of the RTC Interface.

Remarks

Use this function to control settings. The available settings are documented as part of EOS_RTC_SetSettingOptions.

Return Value

EOS_Success when the setting is successfully set, EOS_NotFound when the setting is unknown, EOS_InvalidParameters when the value is invalid.

Parameters

EOS_RTC_SetSetting

Parameter Type And Name Usage Information
EOS_HRTC Handle
const EOS_RTC_SetSettingOptions* Options structure containing the parameters for the operation
