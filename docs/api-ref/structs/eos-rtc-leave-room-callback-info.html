EOS_RTC_LeaveRoomCallbackInfo

EOS API reference page for EOS_RTC_LeaveRoomCallbackInfo

Under 1 min to read

This data structure is part of the RTC Interface.

Remarks

This struct is passed in with a call to EOS_RTC_OnLeaveRoomCallback.

Properties

EOS_RTC_LeaveRoomCallbackInfo

Property Value
EOS_EResult ResultCode This returns: EOS_Success if the channel was successfully left. EOS_AccessDenied if the room name belongs to the Lobby voice system. EOS_UnexpectedError otherwise.
void* ClientData Client-specified data passed into EOS_RTC_LeaveRoomOptions.
EOS_ProductUserId LocalUserId The Product User ID of the user who initiated this request.
const char* RoomName The room the user was trying to leave.
