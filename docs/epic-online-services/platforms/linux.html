EOS SDK for Linux

Reference information for implementing EOS SDK for Linux.

Under 1 min to read

Supported Linux Release Versions

All EOS SDK release versions support any modern Linux version that uses glibc version 2.17 or higher. The table below lists the Linux toolchain version which each EOS SDK release version supports:

EOS SDK Release Version Linux Toolchain Version
EOS SDK 1.11 - 1.16.3 supports v19_clang-11.0.1-centos7 or higher
EOS SDK 1.10.2 - 1.10.3 supports v16_clang-9.0.1-centos7 or higher

Tip: You can call the EOS_GetVersion function to get the EOS SDK version and the version of the platform that your game is compiled against.

Before implementing the SDK with Linux, see the documentation on general guidelines and references for information on platform implementation.

Building EOS on Linux

Linux does not require the use of Linux/eos_Linux_base.h. It is provided to ease implementation of a cross-platform product.

Linking EOS on Linux

Linux requires you to add the shared object library (libEOSSDK-Linux-Shipping.so) to the link command for your project.
