Remarks

Input parameters for the EOS_Connect_LinkAccount function.

Properties

EOS_Connect_LinkAccountOptions

Property Value
int32_t ApiVersion API Version: Set this to EOS_CONNECT_LINKACCOUNT_API_LATEST.
EOS_ProductUserId LocalUserId The existing logged in product user for which to link the external account described by the continuance token.
EOS_ContinuanceToken ContinuanceToken Continuance token from previous call to EOS_Connect_Login.
