Localization in KWS

Supported languages, and how to localize your KWS service.

Under 1 min to read

Languages supported by the KWS Platform

  • The KWS Developer Portal is available only in English.
  • This documentation is available in English, Korean, Japanese, and Simplified Chinese.

Languages supported by KWS services

You can localize KWS services so that all user-facing emails and webviews are in the local language. To do this, you pass the required language code in your call to the KWS API.

KWS supports these languages in its services:
