Remarks

Input parameters for the EOS_Ecom_CopyEntitlementByNameAndIndex function.

Properties

EOS_Ecom_CopyEntitlementByNameAndIndexOptions

Property Value
int32_t ApiVersion API Version: Set this to EOS_ECOM_COPYENTITLEMENTBYNAMEANDINDEX_API_LATEST.
EOS_EpicAccountId LocalUserId The Epic Account ID of the local user whose entitlement is being copied
EOS_Ecom_EntitlementName EntitlementName Name of the entitlement to retrieve from the cache
uint32_t Index Index of the entitlement within the named set to retrieve from the cache.
