EOS_ERTCAudioStatus

EOS API reference page for EOS_ERTCAudioStatus

Under 1 min to read

Remarks

An enumeration of the different audio channel statuses.

Values

EOS_ERTCAudioStatus

Enumerated Value Meaning
EOS_RTCAS_Unsupported Audio unsupported by the source (no devices)
EOS_RTCAS_Enabled Audio enabled
EOS_RTCAS_Disabled Audio disabled
EOS_RTCAS_AdminDisabled Audio disabled by the administrator
EOS_RTCAS_NotListeningDisabled Audio channel is disabled temporarily for both sending and receiving
