EOS_Connect_GetExternalAccountMapping

EOS API reference page for EOS_Connect_GetExternalAccountMapping

Under 1 min to read

This function is part of the Connect Interface.

Remarks

Fetch a Product User ID that maps to an external account ID cached from a previous query.

Return Value

The Product User ID, previously retrieved from the backend service, for the given target external account.

Parameters

EOS_Connect_GetExternalAccountMapping

Parameter Type And Name Usage Information
EOS_HConnect Handle
const EOS_Connect_GetExternalAccountMappingsOptions* Options structure containing the local user and target external account ID.
