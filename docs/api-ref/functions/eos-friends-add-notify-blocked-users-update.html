EOS_Friends_AddNotifyBlockedUsersUpdate

EOS API reference page for EOS_Friends_AddNotifyBlockedUsersUpdate

2 mins to read

This function is part of the Friends Interface.

Remarks

Listen for changes to blocklist for a particular account.

Return Value

A valid notification ID if successfully bound, or EOS_INVALID_NOTIFICATIONID otherwise.

Parameters

EOS_Friends_AddNotifyBlockedUsersUpdate

Parameter Type And Name Usage Information
EOS_HFriends Handle
const EOS_Friends_AddNotifyBlockedUsersUpdateOptions* Options Information about the API version which is being used.
void* ClientData This value is returned to the caller when BlockedUsersUpdateHandler is invoked.
const EOS_Friends_OnBlockedUsersUpdateCallback BlockedUsersUpdateHandler The callback to be invoked when a blocklist changes.

This function enables your callback (the BlockedUsersUpdateHandler parameter) to respond to events that happen on the back-end service. Remember to remove your callback function when it is no longer needed.

Callback Function Information

Because this function is asynchronous, it employs a callback of type EOS_Friends_OnBlockedUsersUpdateCallback to report the results of its operation.

Callback Remarks

Callback for information related to a blocklist update.

Callback Parameters

EOS_Friends_OnBlockedUsersUpdateCallback

Parameter Type And Name Usage Information
