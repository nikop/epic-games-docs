EOS_RTCAudio_OutputDeviceInformation_Release

EOS API reference page for EOS_RTCAudio_OutputDeviceInformation_Release

Under 1 min to read

Remarks

Release the memory associated with EOS_RTCAudio_OutputDeviceInformation. This must be called on data retrieved from EOS_RTCAudio_CopyOutputDeviceInformationByIndex.

See Also

EOS_RTCAudio_OutputDeviceInformation, EOS_RTCAudio_CopyOutputDeviceInformationByIndex

Parameters

EOS_RTCAudio_OutputDeviceInformation_Release

Parameter Type And Name Usage Information
EOS_RTCAudio_OutputDeviceInformation* DeviceInformation - The audio output device's information to release.
