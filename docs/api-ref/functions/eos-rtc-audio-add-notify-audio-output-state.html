EOS_RTCAudio_AddNotifyAudioOutputState

EOS API reference page for EOS_RTCAudio_AddNotifyAudioOutputState

1 min to read

Remarks

Register to receive notifications when audio output state changed. If the returned NotificationId is valid, you must call EOS_RTCAudio_RemoveNotifyAudioOutputState when you no longer wish to have your CompletionDelegate called.

Return Value

Notification ID representing the registered callback if successful, an invalid NotificationId if not

See Also

EOS_INVALID_NOTIFICATIONID, EOS_RTCAudio_RemoveNotifyAudioOutputState
