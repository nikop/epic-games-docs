Support for Epic Developer Resources

If you are a developer using or evaluating the following Epic developer resources: the Developer Portal, Epic Games Store, Epic Online Services (EOS), Kids Web Services (KWS), or other tools and services, see the support options listed below.

Developer Portal

For support articles and help with the Developer Portal, see:

  • The EOS Developer Support website at eoshelp.epicgames.com

    For guidance on how to create tickets, see the section below on Using the EOS Developer Support Website.

Epic Games Store

For support articles and help with the Epic Games Store, see:

  • The Dev Community website: Epic Games section at dev.epicgames.com/~/epic-games-store

Epic Online Services (EOS)

For support articles and help with Epic Online Services (EOS), see:

  • The EOS Developer Support website at eoshelp.epicgames.com

    For guidance on how to create tickets, see the section below on Using the EOS Developer Support Website

For an overview of EOS services, both EOS Game Services and EOS Epic Account Services, see:

  • EOS Overview documentation

Kids Web Services (KWS)

For support articles and help with Kids Web Services (KWS), see:

  • The Epic Online Services Developer Support: Kids Web Services
  • KWS support documentation

Using the EOS Developer Support Website

At eoshelp.epicgames.com, you can ask questions on the website and get answers from the community and EOS Developer Relations Engineers on EOS and the Developer Portal. There are also knowledge-based articles by Epic Games.

To create a new discussion support ticket, you can take either of the following actions on the website:

  • Click CREATE NEW DISCUSSION in the top right to post a public discussion topic that can be read by anyone who visits the forum.
    • Note: As posts will be public - remove sensitive information like IDs, credentials, or non-public product information from your post.
  • Click CREATE PRIVATE DISCUSSION in the top right if you need to include sensitive information in your post.

Support for Other Epic Games Developer Tools and Services

For support with Unreal Engine, UEFN & Creative, Capturing Reality, and MetaHuman, see:

  • The Dev Community website at dev.epicgames.com/community
  • The Unreal Developer Network (UDN) support website
