EOS_Sanctions_GetPlayerSanctionCount

EOS API reference page for EOS_Sanctions_GetPlayerSanctionCount

Under 1 min to read

This function is part of the Sanctions Interface.

Remarks

Fetch the number of player sanctions that have been retrieved for a given player. You must call QueryActivePlayerSanctions first to retrieve the data from the service backend.

Return Value

Number of available sanctions for this player.

See Also

EOS_Sanctions_QueryActivePlayerSanctions, EOS_Sanctions_CopyPlayerSanctionByIndex
