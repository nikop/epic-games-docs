Technical Overview

A technical overview of the Social Overlay, including how to define shortcuts and localization.

3 mins to read

The Social Overlay supports Windows PC, Nintendo Switch, PlayStation, and Xbox console platforms. Support for macOS, Linux, Android, and iOS is planned.

Web Browser Engine

The Social Overlay uses a web browser engine to render the panels and notification pop-ups that the player sees in your game. The web browser engine is a recent fork of WebKit and supports HTML5, JavaScript, CSS3, and Web APIs.

For more information, see the following documentation:

  • Platform Implementation for EOS: for details on the platforms that EOS SDK supports.
  • WebKit at webkit.org: the open source organization that provides the WebKit web browser engine.

Content Delivery Network (CDN)

The Social Overlay uses a Content Delivery Network (CDN) so the player sees the latest update to their social information (for example their friends list) and your games' achievements when the EOS SDK initializes the Social Overlay. This also means we can update the Social Overlay with new features and bug fixes at any time.

You can choose to disable some features of the Social Overlay, so that when the EOS SDK initializes the Social Overlay the player cannot see or use those features.

Note: Epic Games may disable or hide a feature when there are issues that affect the player's experience while playing your game.

Social Overlay from the Epic Games Launcher

A player can open the Social Overlay from the Epic Games Launcher if they select the following icon:

When a player uses the Social Overlay from the Epic Games Launcher, they can view their friends, see which friends are online and accept an invitation to join another player's game. However, they cannot do the following (unless they use the Social Overlay in your game):

  • Request to join another player's game or party.
  • See their own party.
  • Join a public lobby.

Presence

The Social Overlay automatically displays a player's presence information while they play your game. This includes basic details, such as their online status and the name of your game. If you include rich presence strings in your game, the Social Overlay automatically displays this information. Rich presence strings are rich text strings that specify the current action or location of the player, such as “In the Main Menu”.

The Social Overlay automatically displays rich presence strings for the following items:

  • Rich presence strings that you set for the native platform with the native platform's APIs.
  • Rich presence strings that you set for EOS with the EOS SDK.

Note: If you want to display rich presence strings on the native platform (for example, PlayStation 4), you must call the native platform's APIs to set them.

For more information on how to set rich presence strings for EOS, see the Modifying a Rich Text String section of the Presence Interface documentation. For more information on how to set rich presence strings for the native platform, see the native platform's official documentation.

When a player creates their Epic Games account, the language they choose is also the language they see in the Social Overlay. However, you can set the locale for your game, which overrides the language used by the player. See the Localization section of the EOS Overlay Overview documentation for more information.

If the player chooses a language that the Social Overlay does not support, then the Social Overlay uses the next best locale that is supported.

How to Set the Shortcut

Players can open the Social Overlay in your game if they press a shortcut:

  • Windows PC (default): SHIFT+F3
  • Nintendo Switch (recommended): Minus button
  • PlayStation 4, PlayStation 5 (recommended): Touchpad
  • Xbox One, Xbox Series (recommended): View button

Players can use the default Windows PC shortcut (SHIFT+F3) or you can change this shortcut to a different one. If you release your game on console, you must set up the shortcut. We recommend the shortcuts listed above.

To set up a shortcut on Windows PC or console:

  • Call EOS_UI_SetToggleFriendsKey and set the key combination in the EOS_UI_SetToggleFriendsKeyOptions parameter. On Windows PC, you can use any set of modifiers (for example SHIFT) and one other key. On console you can use any button.

For more details on how to set up the shortcut, see the following documentation:

  • Values and parameters for the API: EOS_UI_SetToggleFriendsKey
  • Details on how to set each key combination: EOS_UI_EKeyCombination .
