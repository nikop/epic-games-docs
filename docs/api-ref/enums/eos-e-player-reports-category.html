EOS_EPlayerReportsCategory

EOS API reference page for EOS_EPlayerReportsCategory

1 min to read

This enumerated type is part of the Reports Interface.

Remarks

An enumeration of the different player behavior categories that can be reported.

Values

EOS_EPlayerReportsCategory

Enumerated Value Meaning
EOS_PRC_Invalid Not used
EOS_PRC_Cheating The reported player is cheating
EOS_PRC_Exploiting The reported player is exploiting the game
EOS_PRC_OffensiveProfile The reported player has an offensive profile, name, etc
EOS_PRC_VerbalAbuse The reported player is being abusive in chat
EOS_PRC_Scamming The reported player is scamming other players
EOS_PRC_Spamming The reported player is spamming chat
EOS_PRC_Other The player is being reported for something else
