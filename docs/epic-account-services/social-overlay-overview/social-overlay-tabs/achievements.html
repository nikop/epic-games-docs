Achievements

Details on what a player can do on the Achievements tab of the Social Overlay.

2 mins to read

This documentation assumes that you install the Social Overlay with your game and initialize the EOS SDK when a player starts your game. For more information on how to install and initialize the Social Overlay, see the SDK Integration documentation.

Image: The Achievements tab of the Social Overlay, from Fall Guys (EOS SDK 1.15).

A player can access the Achievements tab of the Social Overlay if you have integrated either Epic Games Store achievements or Epic Online Services achievements into your game.

Note: If you integrate both Epic Games Store achievements and Epic Online Services achievements into your game, players only see Epic Games Store achievements on the Achievements tab of the Social Overlay.

On the Achievements tab of the Social Overlay, a player can see a list of achievements in your game (that aren't hidden from the player), the progress they have made to complete all available achievements, and details on what the achievements are.

For more information on using achievements, see the documentation:

  • Achievements Interface for Epic Online Services achievements.
  • Epic Games Store achievements.

What is displayed?

A player can use the Achievements tab to see the following details on the achievements in your game:

List of Achievements

Achievements are grouped in this order:

  • Completed: Achievements that the player has completed.
  • Locked: Achievements that the player can see, but has not completed.
  • Hidden: Achievements that the player cannot see full details of but can see the achievement title.

Achievement Progress

  • All achievements from your game.
  • Player's total progress: the total number of completed achievements compared to the total number of achievements, progress measured in a percentage, and also represented by a progress bar.
  • Player's complete achievements.
  • If you integrate Epic Games achievements into your game then a player also sees their progress on each achievement (represented by a progress bar).

Achievement Detail

Each achievement shows the following:

  • Icon: The image you set to represent the achievement.
    If you did not set the icon or it fails to load, the player sees a default icon.
  • Title text: The title you set up for the achievement.
    If the achievement is locked, then the player sees a locked symbol next to the title text.
  • Description text: The description you set for the achievement.
  • Player's progress: If you integrate Epic Games achievements into your game then a player also sees their progress on each achievement (represented by a progress bar).
  • Trophy: If you integrate Epic Games achievements into your game then a player also sees a bronze, silver, or gold trophy. For more information on how to set up XP requirements for each trophy tier, see Epic Games Store achievements: Setup Guide.

Note: You can create a different icon, title, and description for achievements that depend on the achievement status: completed, locked, or hidden.
