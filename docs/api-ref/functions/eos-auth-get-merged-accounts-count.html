EOS_Auth_GetMergedAccountsCount

EOS API reference page for EOS_Auth_GetMergedAccountsCount

Under 1 min to read

This function is part of the Auth Interface.

Remarks

Fetch the number of merged accounts for a given logged in account.

Return Value

the number of merged accounts for the logged in account.

Parameters

EOS_Auth_GetMergedAccountsCount

Parameter Type And Name Usage Information
EOS_HAuth Handle
const EOS_EpicAccountId LocalUserId The account ID of a currently logged in account.
