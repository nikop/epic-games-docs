EOS_CustomInvites_RejectRequestToJoinCallbackInfo

EOS API reference page for EOS_CustomInvites_RejectRequestToJoinCallbackInfo

Under 1 min to read

This data structure is part of the CustomInvites Interface.

Remarks

Output parameters for the EOS_CustomInvites_RejectRequestToJoin Function. These parameters are received through the callback provided to EOS_CustomInvites_RejectRequestToJoin

Properties

EOS_CustomInvites_RejectRequestToJoinCallbackInfo

Property Value
EOS_EResult ResultCode The EOS_EResult code for the operation. EOS_Success indicates that the operation succeeded; other codes indicate errors.
void* ClientData Context that was passed into EOS_CustomInvites_RejectRequestToJoin
EOS_ProductUserId LocalUserId Local user declining a request to join
EOS_ProductUserId TargetUserId Target user that sent original request to join
