EOS_RTCAudio_GetOutputDevicesCount

EOS API reference page for EOS_RTCAudio_GetOutputDevicesCount

Under 1 min to read

Remarks

Fetch the number of audio output devices available in the system that are cached locally. The returned value should not be cached and should instead be used immediately with the EOS_RTCAudio_CopyOutputDeviceInformationByIndex function.

Return Value

The number of audio output devices available in the system or 0 if there is an error

See Also

EOS_RTCAudio_CopyOutputDeviceInformationByIndex, EOS_RTCAudio_AddNotifyAudioDevicesChanged
