EOS_Presence_HasPresence

EOS API reference page for EOS_Presence_HasPresence

Under 1 min to read

This function is part of the Presence Interface.

Remarks

Check if we already have presence for a user

Return Value

EOS_TRUE if we have presence for the requested user, or EOS_FALSE if the request was invalid or we do not have cached data

Parameters

EOS_Presence_HasPresence

Parameter Type And Name Usage Information
EOS_HPresence Handle
const EOS_Presence_HasPresenceOptions* Options Object containing properties related to who is requesting presence and for what user
