EOS_Achievements_DefinitionV2_Release

EOS API reference page for EOS_Achievements_DefinitionV2_Release

Under 1 min to read

This function is part of the Achievements Interface.

Remarks

Release the memory associated with EOS_Achievements_DefinitionV2. This must be called on data retrieved from EOS_Achievements_CopyAchievementDefinitionV2ByIndex or EOS_Achievements_CopyAchievementDefinitionV2ByAchievementId.

See Also

EOS_Achievements_DefinitionV2, EOS_Achievements_CopyAchievementDefinitionV2ByIndex, EOS_Achievements_CopyAchievementDefinitionV2ByAchievementId

Parameters

EOS_Achievements_DefinitionV2_Release

Parameter Type And Name Usage Information
EOS_Achievements_DefinitionV2* AchievementDefinition - The achievement definition to release.
