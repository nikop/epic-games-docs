EOS_CustomInvites_SetCustomInvite

EOS API reference page for EOS_CustomInvites_SetCustomInvite

Under 1 min to read

This function is part of the CustomInvites Interface.

Remarks

Initializes a Custom Invite with a specified payload in preparation for it to be sent to another user or users.

Return Value

  • EOS_Success if the operation completes successfully

  • EOS_InvalidParameters if any of the options values are incorrect
