EOS_IntegratedPlatform_UserLoginStatusChangedCallbackInfo

EOS API reference page for EOS_IntegratedPlatform_UserLoginStatusChangedCallbackInfo

1 min to read

This data structure is part of the IntegratedPlatform Interface.

Remarks

Data about which integrated platform and which user that had a login status change and what the login status changed to.

Properties

EOS_IntegratedPlatform_UserLoginStatusChangedCallbackInfo

Property Value
void* ClientData Context that was passed into EOS_IntegratedPlatform_AddNotifyUserLoginStatusChanged
EOS_IntegratedPlatformType PlatformType The integrated platform of the local platform user.
const char* LocalPlatformUserId String version of platform's user id.
EOS_EpicAccountId AccountId The Epic Games Account ID associated with this Integrated Platform's User (if there is one)
EOS_ProductUserId ProductUserId The EOS Product User ID associated with this Integrated Platform's User (if there is one)
EOS_ELoginStatus PreviousLoginStatus The login status prior to this change.
EOS_ELoginStatus CurrentLoginStatus The login status at the time of this notification.
