Localization in KWS

KWS platform-supported languages, and how to localize your KWS service.

Under 1 min to read

Languages supported by the KWS platform

The KWS Developer Portal is available only in English.

  • This documentation is available in English, Korean, Japanese, and Simplified Chinese.

Localizing the Parent Verification service

You can specify the language that KWS uses for parent-facing verification emails and webviews in the PV service. KWS supports these languages:
Chinese (Simplified) zh-Hans Korean ko
Dutch nl Polish pl
English en Portuguese (Brazil) pt-BR
Filipino (Tagalog) tl Portuguese (Portugal) pt
French fr Russian ru
German de Spanish es
Italian it Thai th
Japanese ja Turkish tr

To specify a language, pass the language code in your call to the KWS API.
