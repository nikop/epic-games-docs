EOS_Platform_DesktopCrossplayStatusInfo

EOS API reference page for EOS_Platform_DesktopCrossplayStatusInfo

Under 1 min to read

Remarks

Output parameters for the EOS_Platform_GetDesktopCrossplayStatus function.

Properties

EOS_Platform_DesktopCrossplayStatusInfo

Property Value
EOS_EDesktopCrossplayStatus Status Status for the availability of desktop crossplay functionality. It is recommended to include this value in application logs, and as part of any player-facing error screens to help troubleshooting possible issues.
int32_t ServiceInitResult This field is set when the Status is EOS_DCS_ServiceStartFailed. Possible values for this field are not documented. However, it is recommended to be also included in application logs, and as part of any player-facing error screens.
