EOS_EEcomItemType

EOS API reference page for EOS_EEcomItemType

Under 1 min to read

This enumerated type is part of the Ecom Interface.

Remarks

An enumeration defining the type of catalog item. The primary use is to identify how the item is expended.

Values

EOS_EEcomItemType

Enumerated Value Meaning
EOS_EIT_Durable This entitlement is intended to persist.
EOS_EIT_Consumable This entitlement is intended to be transient and redeemed.
EOS_EIT_Other This entitlement has a type that is not currently intended for an in-game store.
