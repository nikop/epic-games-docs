EOS_Mods_UpdateModCallbackInfo

EOS API reference page for EOS_Mods_UpdateModCallbackInfo

1 min to read

This data structure is part of the Mods Interface.

Remarks

Output parameters for the EOS_Mods_UpdateMod Function. These parameters are received through the callback provided to EOS_Mods_UpdateMod

Properties

EOS_Mods_UpdateModCallbackInfo

Property Value
EOS_EResult ResultCode Result code for the operation. EOS_Success is returned if the request to update was successful, otherwise one of the error codes is returned.
EOS_EpicAccountId LocalUserId The Epic Account ID of the user for which mod update was requested
void* ClientData Context that is passed into EOS_Mods_UpdateMod
const EOS_Mod_Identifier* Mod Mod for which update was requested
