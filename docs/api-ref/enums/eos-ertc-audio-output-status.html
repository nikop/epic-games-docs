EOS_ERTCAudioOutputStatus

EOS API reference page for EOS_ERTCAudioOutputStatus

Under 1 min to read

Remarks

An enumeration of the different audio output device statuses.

Values

EOS_ERTCAudioOutputStatus

Enumerated Value Meaning
EOS_RTCAOS_Idle The device is not in used right now (e.g: you are alone in the room). In such cases, the hardware resources are not allocated.
EOS_RTCAOS_Playing Device is in use
EOS_RTCAOS_Failed Something failed while trying to use the device
