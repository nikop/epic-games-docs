Remarks

This struct is used to call EOS_RTCData_SendData.

Properties

EOS_RTCData_SendDataOptions

Property Value
int32_t ApiVersion API Version: Set this to EOS_RTCDATA_SENDDATA_API_LATEST.
EOS_ProductUserId LocalUserId The Product User ID of the user trying to request this operation.
const char* RoomName The room this event is registered on.
uint32_t DataLengthBytes The size of the data to be sent to the other participants. Max value is EOS_RTCDATA_MAX_PACKET_SIZE.
const void* Data The data to be sent to the other participants
