EOS_IntegratedPlatform_UserPreLogoutCallbackInfo

EOS API reference page for EOS_IntegratedPlatform_UserPreLogoutCallbackInfo

1 min to read

This data structure is part of the IntegratedPlatform Interface.

Remarks

Data passed to the application in the EOS_IntegratedPlatform_OnUserPreLogoutCallback function. This contains which user and associated Integrated Platform that was detected as logged-out.

Properties

EOS_IntegratedPlatform_UserPreLogoutCallbackInfo

Property Value
void* ClientData Context that was passed into EOS_IntegratedPlatform_SetUserPreLogoutCallback
EOS_IntegratedPlatformType PlatformType The integrated platform the local user logged-out of.
const char* LocalPlatformUserId String version of platform-dependent user id.
EOS_EpicAccountId AccountId The Epic Games Account ID associated with this Integrated Platform's User (if there is one)
EOS_ProductUserId ProductUserId The EOS Product User ID associated with this Integrated Platform's User (if there is one)
