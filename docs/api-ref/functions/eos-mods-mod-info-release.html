EOS_Mods_ModInfo_Release

EOS API reference page for EOS_Mods_ModInfo_Release

Under 1 min to read

This function is part of the Mods Interface.

Remarks

Release the memory associated with an EOS_Mods_ModInfo structure and its sub-objects. This must be called on data retrieved from EOS_Mods_CopyModInfo.

See Also

EOS_Mods_ModInfo, EOS_Mods_CopyModInfo

Parameters

EOS_Mods_ModInfo_Release

Parameter Type And Name Usage Information
EOS_Mods_ModInfo* ModInfo the info structure to be release
