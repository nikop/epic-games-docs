EOS_KWS_RequestPermissionsCallbackInfo

EOS API reference page for EOS_KWS_RequestPermissionsCallbackInfo

Under 1 min to read

This data structure is part of the KWS Interface.

Remarks

Output parameters for the EOS_KWS_RequestPermissionsOptions Function. These parameters are received through the callback provided to EOS_KWS_RequestPermissions

Properties

EOS_KWS_RequestPermissionsCallbackInfo

Property Value
EOS_EResult ResultCode The EOS_EResult code for the operation. EOS_Success indicates that the operation succeeded; other codes indicate errors.
void* ClientData Context that was passed into EOS_KWS_RequestPermissions
EOS_ProductUserId LocalUserId Local user requesting new permissions
