The KWS API

An overview of the KWS API.

Under 1 min to read

KWS uses a RESTful API over HTTP. It works with any platform/language capable of making POST requests over HTTPS.

See the KWS API reference here.

NOTE: If you expect your app to have more than 10,000 monthly active users that are under 16, contact us.

For details of how to call the API after configuring your PV service settings, see Integrate with the KWS API.
