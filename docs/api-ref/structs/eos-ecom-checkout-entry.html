EOS_Ecom_CheckoutEntry

EOS API reference page for EOS_Ecom_CheckoutEntry

Under 1 min to read

This data structure is part of the Ecom Interface.

Remarks

Contains information about a request to purchase a single offer. This structure is set as part of the EOS_Ecom_CheckoutOptions structure.

Properties

EOS_Ecom_CheckoutEntry

Property Value
int32_t ApiVersion API Version: Set this to EOS_ECOM_CHECKOUTENTRY_API_LATEST.
EOS_Ecom_CatalogOfferId OfferId The ID of the offer to purchase
