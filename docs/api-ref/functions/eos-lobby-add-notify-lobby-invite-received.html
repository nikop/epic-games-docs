EOS_Lobby_AddNotifyLobbyInviteReceived

EOS API reference page for EOS_Lobby_AddNotifyLobbyInviteReceived

2 mins to read

This function is part of the Lobby Interface.

Remarks

Register to receive notifications about lobby invites sent to local users.

must call RemoveNotifyLobbyInviteReceived to remove the notification

Return Value

handle representing the registered callback

Parameters

EOS_Lobby_AddNotifyLobbyInviteReceived

Parameter Type And Name Usage Information
EOS_HLobby Handle
const EOS_Lobby_AddNotifyLobbyInviteReceivedOptions* Options Structure containing information about the request.
void* ClientData Arbitrary data that is passed back to you in the CompletionDelegate.
const EOS_Lobby_OnLobbyInviteReceivedCallback NotificationFn A callback that is fired when a a notification is received.

This function enables your callback (the NotificationFn parameter) to respond to events that happen on the back-end service. Remember to remove your callback function when it is no longer needed.

Callback Function Information

Because this function is asynchronous, it employs a callback of type EOS_Lobby_OnLobbyInviteReceivedCallback to report the results of its operation.

Callback Remarks
