EOS_LobbySearch_GetSearchResultCount

EOS API reference page for EOS_LobbySearch_GetSearchResultCount

Under 1 min to read

This function is part of the Lobby Interface.

Remarks

Get the number of search results found by the search parameters in this search

Return Value

return the number of search results found by the query or 0 if search is not complete
