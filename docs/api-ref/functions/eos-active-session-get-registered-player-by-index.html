EOS_ActiveSession_GetRegisteredPlayerByIndex

EOS API reference page for EOS_ActiveSession_GetRegisteredPlayerByIndex

Under 1 min to read

This function is part of the Sessions Interface.

Remarks

EOS_ActiveSession_GetRegisteredPlayerByIndex is used to immediately retrieve individual players registered with the active session.

Return Value

the product user ID for the registered player at a given index or null if that index is invalid
