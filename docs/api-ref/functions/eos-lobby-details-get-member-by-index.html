EOS_LobbyDetails_GetMemberByIndex

EOS API reference page for EOS_LobbyDetails_GetMemberByIndex

Under 1 min to read

This function is part of the Lobby Interface.

Remarks

EOS_LobbyDetails_GetMemberByIndex is used to immediately retrieve individual members registered with a lobby.

Return Value

the product user ID for the registered member at a given index or null if that index is invalid
