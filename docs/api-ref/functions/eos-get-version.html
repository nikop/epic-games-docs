EOS API reference page for EOS_GetVersion

Remarks

Get the version of the compiled EOSSDK binary MAJOR.MINOR.PATCH-CHANGELIST#-PLATFORMSDKVERSION It is the version available at runtime It should match the version of the headers being used

Return Value

the version of the compiled EOSSDK binary

Parameters

EOS_GetVersion

This function does not take any parameters.
