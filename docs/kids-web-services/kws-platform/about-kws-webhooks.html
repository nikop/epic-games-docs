KWS Webhooks

An overview of how KWS uses webhooks to notify your system of events.

Under 1 min to read

KWS uses webhooks to notify your system of important service events. This requires some configuration on your side to receive the webhook calls:

  • In the Developer Portal, you provide the URL to which KWS will send the webhook call.
  • Using the secret code that KWS provides, you need to generate a signature. This is matched against KWS' signature to confirm the webhook call is really coming from KWS.

For details, see Configure the PV service webhook.
