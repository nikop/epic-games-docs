Remarks

Flags describing the type of a remote client

Values

EOS_EAntiCheatCommonClientType

Enumerated Value Meaning
EOS_ACCCT_ProtectedClient An ordinary player that requires anti-cheat client protection to play
EOS_ACCCT_UnprotectedClient The player does not need the anti-cheat client to play because of their platform or other factors
EOS_ACCCT_AIBot The client is an AI bot, not an actual human
