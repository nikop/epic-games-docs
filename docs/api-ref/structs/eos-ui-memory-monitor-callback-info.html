EOS_UI_MemoryMonitorCallbackInfo

EOS API reference page for EOS_UI_MemoryMonitorCallbackInfo

Under 1 min to read

This data structure is part of the UI Interface.

Remarks

A structure representing a memory monitoring message.

Properties

EOS_UI_MemoryMonitorCallbackInfo

Property Value
void* ClientData Context that was passed into EOS_UI_AddNotifyMemoryMonitor
const void* SystemMemoryMonitorReport This field is for system specific memory monitor report. If provided then the structure will be located in eos_<Platform>ui.h The structure will be named EOS<Platform>_MemoryMonitorReport.
