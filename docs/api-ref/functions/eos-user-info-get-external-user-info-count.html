EOS_UserInfo_GetExternalUserInfoCount

EOS API reference page for EOS_UserInfo_GetExternalUserInfoCount

Under 1 min to read

This function is part of the UserInfo Interface.

Remarks

Fetch the number of external user infos that are cached locally.

Return Value

The number of external user infos, or 0 if there is an error
